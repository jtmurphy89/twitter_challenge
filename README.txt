To execute all programs at once, type:

user: source run_script.sh

This will put the outputs of part1, part2 and part3 into their respective outputs.txt files. 

For part 2: polarity score given by a number n in [-1,1], where n < 0 corresponds to negative polarity, n = 0 neutral polarity ad n > 0 positive polarity. Similarly, a sentiment score is given as a number in the interval [0,1] with scores close to 0 representing more objective sentiment and scores close to 1 respresenting subjective sentiment.
